## 📱 About

TraVQR (QR Code Transaction Verification) is a mobile banking application designed to combat QR code fraud through real-time verification and OCR-based overlay detection. Developed as a Final Year Project, this application addresses the critical security challenges in Malaysia's rapidly growing digital payment ecosystem.

**Project Type:** Final Year Project (FYP)  
**Program:** Bachelor of Computer Science with Honours (Information Security and Assurance)  

## 🚨 Problem Statement

The widespread adoption of QR codes for digital payments in Malaysia has introduced new security vulnerabilities:

- **QR Code Overlay Attacks:** Fraudsters place fake QR codes over legitimate merchant codes
- **Lack of Verification:** Users cannot verify recipient details before payment
- **Visual Deception:** Current apps only decode QR data without checking for tampering
- **Financial Loss:** Victims unknowingly transfer money to scammers

According to reports, QR code scams have increased significantly in Malaysia, with cases of fake codes being pasted over real ones at restaurants, parking lots, and retail stores.

## ✨ Features

### Core Functionality
- 🔍 **QR Code Scanner** - Scan and decode payment QR codes
- ✅ **Real-time Verification** - Cross-check recipient information with database
- 🎯 **OCR Detection** - Identify overlay attacks using image recognition
- ⚠️ **Fraud Alerts** - Warn users of mismatches and suspicious activity
- 📊 **Transaction History** - Complete audit trail of all payments

### Security Features
- 🔐 **Device Authentication** - Register and manage trusted devices
- 🔒 **Encrypted Storage** - Secure password hashing with SCRYPT
- 📝 **Activity Logging** - Comprehensive verification and transaction logs
- 🛡️ **Spatial Analysis** - Detect physical QR code overlays

### User Management
- 👤 **Account Registration** - Secure user onboarding
- 🔑 **Login/Logout** - Authentication with device verification
- 💰 **Balance View** - Real-time account balance display
- 💸 **Money Transfer** - Secure peer-to-peer payments
